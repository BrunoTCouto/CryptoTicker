# CryptoTicker

Simple Project that fetches *bitcoin* and *ethereum* prices from <a href="https://www.bitcointrade.com.br/pt-BR/" target="_blank">BitcoinTrade</a> and display on the oled screen

## Parts
| Quantity |       Part       |
|:--------:|:----------------:|
|     1    |   Amica NodeMcu  |
|     1    |      SSD1306     |
|     1    | 3D Printed Parts |
